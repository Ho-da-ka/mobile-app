## ADDED Requirements

### Requirement: Admin mobile login via HTTP Basic
系统 MUST 支持管理角色在小程序端通过 HTTP Basic 完成登录并访问受保护接口。

### Requirement: Student management CRUD in mobile admin
系统 MUST 提供学员管理完整 CRUD（列表/详情/新增/编辑）能力。

### Requirement: Course management CRUD in mobile admin
系统 MUST 提供课程管理完整 CRUD（列表/详情/新增/编辑）能力。

### Requirement: Draft persistence with explicit cancel discard
系统 MUST 提供草稿恢复能力，并明确“取消”行为为放弃当前编辑。

### Requirement: Consistent error handling and UX feedback
系统 MUST 对关键错误码提供一致且可读的反馈。
