## ADDED Requirements

### Requirement: Booking capacity control
系统 MUST 对课程预约执行容量限制，规则为固定容量 + 先到先得。

### Requirement: Booking concurrency safety
系统 MUST 在并发请求下保证预约结果正确，不超卖、不重复。

### Requirement: Parent manual check-in and attendance mapping
系统 MUST 支持家长手动签到，并将结果映射到考勤体系。

### Requirement: In-app message lifecycle
系统 MUST 提供站内消息列表与已读状态流转能力。
