## ADDED Requirements

### Requirement: Parent role access boundary
系统 MUST 支持家长角色访问自身授权范围内的数据，并严格执行可见性隔离。

### Requirement: Parent course browsing
系统 MUST 支持家长查看可预约课程信息。

### Requirement: Parent booking lifecycle
系统 MUST 支持家长完成预约与取消预约，且遵循容量与关系规则。

### Requirement: Parent manual check-in
系统 MUST 支持家长对有效预约进行手动签到并同步考勤结果。

### Requirement: Parent fitness visibility
系统 MUST 支持家长查看其可见学员的体测记录。

### Requirement: Parent in-app messaging
系统 MUST 支持家长查看站内消息并标记已读。
